About this website

I'm Alexandre Kings, the person behind PhysicsMonkey.com. I like creating dynamic visualisations about concepts in Quantum Mechanics - graphs are always nicer when they respond to your inputs!

This website is written in HTML, CSS and JavaScript. Dynamic 2D graphs are programmed using the JavaScript Canvas API, and 3D graphs are made with three.js.

This website's wonderful logos were designed by Jonathan Kings.
